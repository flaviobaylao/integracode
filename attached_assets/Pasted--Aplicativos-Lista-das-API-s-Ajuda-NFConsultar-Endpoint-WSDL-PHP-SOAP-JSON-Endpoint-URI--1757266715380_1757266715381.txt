NFConsultar
Endpoint
WSDL
PHP SOAP
JSON
Endpoint URI
https://app.omie.com.br/api/v1/produtos/nfconsultar/
Métodos
Veja abaixo a documentação de cada método disponível para este serviço.

ConsultarNF
Consulta um Nota Fiscal

Parâmetros:
 	nfChave	nfChave	
Chave de pesquisa da Nota Fiscal 
Retorno nfCadastro: Dados da Nota Fiscal&nbsp;
Exemplo:
{
  "nCodNF": 0,
  "nNF": "1"
}
ListarNF
Listar as Notas Fiscais cadastradas.

Parâmetros:
 	nfListarRequest	nfListarRequest	
Solicitação de Listagem de Notas Fiscais
Retorno nfListarResponse: Resposta da listagem de Notas Fiscais
Exemplo:
{
  "pagina": 1,
  "registros_por_pagina": 20,
  "apenas_importado_api": "N",
  "ordenar_por": "CODIGO"
}
